# PDVR Readme

### IMPORTANT NOTE: PDVR requires the latest release of emawind84's questzdoom fork.

Get that here: https://github.com/emawind84/QuestZDoom/releases

## Controls

Before we begin, read the controls here! PDVR's controls are tailored for what I would assume to be the average keys bound for a player who makes use of the dual wielding in any noteable way.

Fire - Fires your gun. Go figure!

Altfire - Reloads your gun. More mods use altfire than the actual reload button, so I'm using it. Recommended bind: right grip + right trigger

Offhand Fire - Differs based on gun. For the pump shotgun, this grips the pump so you can pump it with your offhand. For the chaingun, it spins the barrels up. For the rifles, it toggles the fireselector. Recommended bind: left trigger

Offhand Altfire - Only used by the inventory manager. If you don't have this set, and don't want to set it, you can just turn off encumberance in the settings. Recommended bind: right grip + left trigger

### Other special functions:

Holding use when encumberance is enabled will show you what all you have in your inventory, and how encumbered you are in total.

With your main hand empty, FIRE will do a simple punch, ALTFIRE will open and close the inventory manager, and OFFHAND FIRE will take off your armor, if you're wearing any.

Note that some weapons require you to grip them with both hands in order to fire them, and others will be much more difficult to control when wielded with one hand. Not that I'll stop you from getting yourself killed, I thrive off of doom players' suffering.

## Misc information

Enemy bullet weapons will *fuck your shit up* if you aren't wearing armor. Just a single shotgun shell or SMG spray to the fleshy bits *will kill you.* If you don't like this, you can turn player damage down in the settings, but try not to shy away from a challenge if you can help it ;3c
